Greenbubble

A simple SMS integration project using Airtable, Twilio, and PHP.

Greenbubble allows you to receive and respond to incoming SMS messages through Twilio by using custom trigger keywords stored in an Airtable base.

Simply send an SMS with a pre-programmed trigger keyword and the system will automatically reply with the corresponding message from your Airtable base.

Getting Started

  1. Clone the Greenbubble repository to your local environment.
  2. Set up your Airtable base with trigger keywords and response messages.
  3. Deploy Greenbubble to your preferred hosting provider or serverless platform.
  4. Update your Twilio and Slacks webhooks to work with your deployed Greenbubble instance.

Features

  • Easy integration with Twilio and Airtable.
  • Automatic response to incoming SMS messages based on trigger keywords.
  • Dynamic content management through Airtable.
  • Support for deployment on various hosting providers and serverless platforms.
